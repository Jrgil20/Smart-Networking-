# AI Usage Tracking

Esta carpeta almacena el historial de uso de inteligencia artificial para el proyecto Smart Networking.
Cada archivo documenta una sesión de trabajo con IA, incluyendo fecha, objetivos, acciones realizadas, decisiones tomadas y resultados.

Usa este directorio para auditar cambios asistidos por IA, verificar por qué se hicieron actualizaciones y conservar el contexto de cada sesión.

## Archivos en esta carpeta

- `ai-usage-05082926.md` — Resumen de la sesión de planificación y documentación general del hackathon.
- `ai-usage-05092926.md` — Resumen de la sesión actual de limpieza de vulnerabilidades y actualización de dependencias.
